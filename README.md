# 🔄 Text-Fixer (הפוך על הפוך)

סקריפט פייתון קליל שרץ ברקע וחוסך תסכול יומיומי: תיקון מיידי של טקסט שהוקלד בטעות כשהמקלדת הייתה בשפה הלא נכונה (עברית במקום אנגלית או להפך). 

במקום למחוק ולהקליד הכל מחדש, פשוט מסמנים את הטקסט, לוחצים על קיצור מקלדת – והטקסט מומר ומוחלף אוטומטית במקום.

## ✨ תכונות עיקריות
* **תיקון אוטומטי:** אין צורך להגדיר לאיזו שפה להמיר, התוכנית מזהה לבד.
* **תמיכה בטקסט מעורב (Chunks):** מטפל בצורה חלקה במשפטים המכילים מעברים בין אנגלית לעברית, תוך שמירה על מספרים וסימני פיסוק ללא שינוי.
* **אוניברסלי:** עובד בכל תוכנה במחשב (דפדפן, Word, סביבות פיתוח, צ'אטים ועוד) בזכות שימוש בלוח הגזירים של מערכת ההפעלה.

## 🚀 דוגמה
נניח שהקלדת בטעות משפט מעורב בלי לשים לב לשפת המקלדת:

> `tbh tuvc aueuks uau,v בםככןק 4 pgnho chuo`

כל מה שצריך לעשות זה **לסמן את הטקסט** וללחוץ על `Ctrl + Shift + A`.
הטקסט ימחק ויוקלד מחדש אוטומטית בצורה התקינה:

> `אני אוהב שוקולד עוגת coffee 4 פעמים ביום`

## 🛠️ מבנה הפרויקט
הפרויקט מחולק לשני קבצים לשמירה על סדר וארכיטקטורה נכונה:
* `converter.py` - מכיל את מחלקת `LanguageConverter` והלוגיקה של מפות ההמרה ופירוק הטקסט.
* `main.py` - סקריפט ההרצה הראשי שמנהל את האזנת המקלדת (Hotkeys) והעבודה מול לוח הגזירים.
